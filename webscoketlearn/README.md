### 实现２　基于web端和Console 的聊天工具

Step1 开启webchat服务端

```
cd webscoketlearn/examples/chatio
npm install 
cd chat
npm install 
npm start
```
！若报错,要关闭相关端口的进程

Step2 运行Console客户端

```
./sio_console_demo
```
